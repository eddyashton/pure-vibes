# Laser Sliders

Block-sliding laser puzzle game built with pure HTML, CSS, and JavaScript.

## How to play

Open `index.html` in any modern browser.

1. Select a movable block.
2. Click a highlighted adjacent square to slide it one cell.
3. Use mirrors to bounce the laser into the glowing target.
4. Crates can be repositioned, but they stop the beam completely.

## Features

- Four handcrafted levels with deterministic scrambles
- Live laser preview after every move
- Mirror and blocker pieces with keyboard or mouse control
- Local-storage level unlock progress
- No dependencies and no build step
