# Snake

Classic grid-based Snake game — pure HTML/CSS/JS, zero dependencies, no build step.

## How to play

Open `index.html` in any modern browser.

| Input | Action |
|---|---|
| Arrow keys / WASD | Change direction |
| P / Space | Pause / resume |
| Enter | Start / restart |
| Swipe (mobile) | Change direction |

## Features

- 20 × 20 grid rendered on a `<canvas>`
- Score tracking with **local-storage high score**
- 10 speed levels — one level gained every 5 points (caps at ~55 ms/tick)
- Smooth food-pulse animation and gradient snake body
- Snake eyes that face the current direction
- Death flash + game-over overlay
- Touch / swipe support for mobile
